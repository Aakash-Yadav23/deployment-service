# Express TypeScript Template

A template for building RESTful APIs using Express and TypeScript.

## Features

- TypeScript support
- Express.js framework
- Mongoose
- Docker
- Docker Compose 
- Environment variable management with `dotenv`
- Basic project structure
- Simple example API

## Prerequisites

- Node.js (v14 or later)
- npm or Yarn
